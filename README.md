## 🎯 Technical Focus Areas

### ☁️ Cloud Security & Observability
Logging, monitoring, and alerting are where I spend most of my time. Recent work includes CloudTrail and CloudWatch-based detection, event-driven automation, and automated response workflows using Lambda and EventBridge.

### ⚙️ Infrastructure & Automation
I prefer repeatable, scripted setups over one-off configurations. If something can be defined as infrastructure or automated safely, that's the direction I take — Terraform, CloudFormation, and Python scripts over manual clicks.

### 🔐 Security Fundamentals
IAM design, least-privilege access, network boundaries, and understanding how misconfigurations actually get exploited — not just documented. Zero Trust isn't a buzzword here, it's a design constraint.

### 🤖 AI + Security
Exploring where AI can realistically improve detection, triage, and secure development workflows — while staying clear-eyed about its limits and failure modes.
